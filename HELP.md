# Справка по языковому процессору

Данная программа предназначена для лексического анализа входного языка.

## 1. Алфавит языка
* Заглавные и строчные латинские буквы (`A-Z`, `a-z`)
* Цифры (`0-9`)
* Операторы: `+`, `-`, `*`, `/`, `=`
* Разделители: `;`, `(`, `)`, `{`, `}`

## 2. Ключевые слова
`if`, `else`, `while`, `int`, `float`, `return`.

## 3. Инструкция по работе
1. Напишите код в окне редактора или откройте файл (**Ctrl+O**).
2. Нажмите кнопку **"Пуск"** (F5) для запуска анализа.
3. Результаты (список лексем или ошибки) появятся в нижней таблице.

## 4. Горячие клавиши
* **Ctrl+N** — Новый файл
* **Ctrl+S** — Сохранить
* **F1** — Вызов справки
